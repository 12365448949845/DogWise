========================================
会议白板助手 - 一键启动脚本使用说明
========================================

✅ 脚本已创建完成并测试通过！

📁 文件位置：
- start-app.ps1  (PowerShell脚本 - 推荐使用)
- start-app.bat  (批处理脚本 - 备选方案)

🚀 快速启动方法：

方法1：PowerShell脚本（推荐）
  1. 右键点击 start-app.ps1
  2. 选择"使用 PowerShell 运行"
  3. 等待5-10秒，浏览器自动打开

方法2：批处理脚本
  1. 双击 start-app.bat
  2. 等待服务启动完成
  3. 浏览器自动打开

🌐 访问地址：
- 前端: http://localhost:5173
- 后端: http://localhost:3001

⚙️ 脚本功能：
✅ 自动检测端口占用并释放
✅ 同时启动前端和后端服务
✅ 服务就绪后自动打开浏览器
✅ 自动检查依赖是否安装
✅ 详细的启动日志输出

⏹️ 停止服务：
- 按 Ctrl+C 或关闭窗口

📋 日志文件：
- frontend.log (前端日志)
- backend.log (后端日志)

⚠️ 注意事项：
1. 确保已安装 Node.js
2. 确保已运行过 npm install
3. 如遇问题，查看日志文件
